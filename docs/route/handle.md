# `handle`

导出一个句柄允许您使用 [`useMatches`][use-matches] 钩子创建应用程序约定。您可以在上面放置任何您想要的值：

```tsx
export const handle = {
  its: "all yours",
};
```

这几乎总是与 `useMatches` 一起使用。要了解您可以用它做什么，请参阅 [`useMatches`][use-matches] 以获取更多信息。
